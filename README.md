# Herramientas Gratis - Multi-language Online Tools

Free online tools targeting Spanish and Portuguese speaking markets.

## Features
- 3 languages: English, Spanish, Portuguese
- IP Address Lookup
- Password Generator
- Word Counter
- QR Code Generator
- Responsive design
- Google AdSense ready

## Tech Stack
- Next.js 15
- Tailwind CSS 4
- TypeScript

## Deploy to Vercel
1. Push to GitHub
2. Import repo in Vercel
3. Deploy automatically
